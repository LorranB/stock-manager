# Estoque Manager

An automated **pricing intelligence desktop application**, designed to streamline inventory analysis for smartphone and tablet retailers — built using **Python, Selenium, Tkinter, and data processing tools**.

This project was created as a **personal experiment and practical business solution**, developed by **Bruno Lorran Sanches do Vale** in collaboration with **ChatGPT**, to explore and enhance skills in **automation, desktop application development, web scraping, and business-oriented software design**.

---

## 🧭 Overview

The Estoque Manager was designed to simplify one of the most repetitive tasks in retail operations: **pricing products competitively and efficiently**.

By combining **inventory reading, competitor analysis, and automated report generation**, the application transforms supplier stock lists into actionable pricing insights within minutes.

> 💡 This project was developed as both a **learning journey** and a real-world productivity tool, demonstrating how automation can eliminate manual work and improve decision-making.

## ✨ Features

* 📄 Automatic supplier PDF inventory reading
* 🌐 Competitor price analysis through Boadica
* 🧠 Intelligent selling price recommendations
* 📊 Automated pricing report generation
* 📱 WhatsApp-ready product catalog creation
* 📈 Excel export for operational analysis
* 🕒 Historical price tracking and monitoring
* 🖥️ Intuitive desktop interface with progress tracking
